# TaskFlow

TaskFlow, kullanıcıların görevlerini düzenli bir şekilde yönetmelerine, sürükle-bırak yöntemiyle kategorize etmelerine ve verimli bir iş akışı oluşturmalarına olanak tanıyan, modern bir görev yönetimi uygulamasıdır.

## 🚀 Proje Hakkında
Bu proje, yazılım geliştirme disiplinimi güçlendirmek, veritabanı mimarisi kurgulamak ve ön yüz (front-end) ile arka yüz (back-end) entegrasyonu konularında profesyonel pratik yapmak amacıyla geliştirilmiştir. Uygulama, kullanıcıların kendi panolarını oluşturmasına, görevlerini yönetmesine ve iş süreçlerini izlemesine imkan tanır.

## 🛠️ Kullanılan Teknolojiler

### Front-end
- **React (Vite)**
- **Tailwind CSS** (Stil yönetimi)
- **dnd-kit** (Sürükle-bırak fonksiyonelliği)
- **TanStack Query** (Veri çekme ve state yönetimi)

### Back-end
- **Node.js & Express**
- **MySQL** (Veritabanı)
- **Prisma** (ORM - Veritabanı etkileşimi)
- **JSON Web Token (JWT)** (Güvenli kimlik doğrulama)



## 📋 Geliştirme Yol Haritası
- [ ] **Veritabanı Şeması:** MySQL tablolarının (Users, Boards, Columns, Cards) tasarımı.
- [ ] **Back-end:** REST API uç noktalarının oluşturulması ve JWT auth entegrasyonu.
- [ ] **Front-end:** Temel layout ve Dashboard sayfasının kodlanması.
- [ ] **Etkileşim:** `dnd-kit` ile sürükle-bırak mantığının kurulması.
- [ ] **Optimizasyon:** Performans iyileştirmeleri ve hata yönetimi (Error handling).
